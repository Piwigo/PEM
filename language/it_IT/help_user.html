Guida del contribuente


Aggiungere un'estensione

Per contribuire al progetto, la prima tappa consiste a creare un'estensione. Cliccare su Aggiungere un'estensione nel menù.

  1. Dare un nome
  2. Scegliere una o più categorie
  3. Riempire la descrizione. È possibile usare le estensioni multilingue per adattare il contenuto alla lingua scelta dal visitatore.

Sulla pagina di visualizzazione dell'estensione, una serie di pulzanti danno accesso a alterttante azioni.

  1. modificare le informazioni dell'estensione
  2. cancellare l'estensione.
  3. gestire i links esterni
  4. illustrare l'estensione con un'immagine
  5. aggiungere una revisione
Aggiungere una revisione

Ogni revisione corresponde a l'evoluzione di un'estensione. Ci sono dunque più revisioni per estensione. Per creare una nuova revisione, cliccare sul pulzante "aggiungere una revisione" nella pagina di visualizzazione dell'estensione

  1. Dare un numero alla revisione, per esempio "2.0"
  2. Selezionare l'archivio corrispondente, al formato ZIP
  3. Scegliere una lista di versioni compatibile con questa revisione dell'estensione. Per esempio, se il software si chiama "Piwigo" e l'estensione "dynamic menu", allora bisogna dichiarare la revisione "2.0" di "dynamic menu" compatible con le versioni 2.0.0, 2.0.1 et 2.0.2 di "Piwigo".
  4. Indicare la lista delle modifiche apportate con la nuova revisione. È possibile usare l'estensione multilingua per adattare il contenuto alla lingua scelta dal visitatore.
Modificare una revisione
  1. Modificare la revisione
  2. Cancellare la revisione
Attenzione :
  • L’archivio di una revisione non può essere modificato. Se sorge un'errore nel archivio, bisogna cancellare la revisione e crearla di nuovo.
  • Appena avete pubblicato e comunicato la vostra contribuzione, non cancellatela. Create piuttosto una nuova revisione. Infatti, gli utenti che usano la vostra contrubuzione potrebbero credere a torto di usare l'ultima versione.
Descrizioni multilingue

Le descrizioni multilingue si fanno tra i tag [lang=xx] e [/lang], dove xx è il codice della lingua (per esempio fr, en, es, it, ....) Esistono anche due tag speciali : [lang=default] e [lang=all]

Esempio d'uso :

Se il contenuto non è disponibile nella lingua scelta dal visitatore, sarà usato l'inglese lingua di default di PEM.
