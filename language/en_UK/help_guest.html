Help for the PEM user


Project Extension Manager

PEM is a PHP/MySql script under GNU license that allows registered users to provide everybody with extensions pertaining to various categories.

Find an extension

To find the extension you're looking for, you can either:

  • Use the filters

    To make the search easier, 3 filtering criterias are available

    • Category
    • Version
    • Author
  • Use the search tool

    Use keywords to find the desired extension

  • Select a version

    Having found the desired extension, be sure to check it is compatible with the version you're currently using before downloading.

Handing over a contribution

To contribute, you only need an account with the extensions manager

  • Login

    If there is a FluxBB forum on the website, and you are registered with it, you can login with your forum user and password.

  • Create an account

    Just click on Register and fill in the required information.
