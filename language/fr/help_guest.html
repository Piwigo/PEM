Aide à l'utilisation de PEM


Projet Extension Manager

PEM est un script PHP/MySql sous licence GNU qui permet aux utilisateurs enregistrés de mettre à disposition de tous des extensions à travers diverses catégories.

Trouver une extension

Pour trouver l'extension dont vous avez besoin, vous pouvez :

  • Utiliser les filtres

    Pour faciliter les recherches d'extension, on peut filtrer suivant 3 critères

    • Catégorie
    • Version
    • Auteur
  • Utiliser la recherche

    Vous pouvez utiliser des mots clés pour trouver l'extension désirée

  • Choisir sa version

    Quand vous avez trouvé l'extension qui vous convient, avant de télécharger, vérifier qu'elle est compatible avec la version que vous utilisez.

Mettre à disposition une Contribution

Pour contribuer, il suffit d'avoir un compte sur le gestionnaire d'extension

  • S'identifier

    Si le site comporte un forum FluxBB et que vous êtes inscrit sur le Forum, vous pouvez vous connecter avec votre pseudo et votre mot de passe du Forum.

  • Créer un compte

    Pour cela cliquer sur S'enregistrer et remplir les informations demandées.
