Ayuda al usuario de PEM


Project Extension Manager (PEM)

Project Extension Manager (o PEM) es un script PHP / MySql bajo licencia GNU que permite a los usuarios registrados poner a disposición extensiones a través de diversas categorías.

Encontrar una extensión

Para encontrar la extensión que estás buscando puedes:

  • Utilizar los filtros

    Para hacer búsquedas de forma más fácil, tienes disponibles tres criterios

    • Categoría
    • Versión
    • Autor
  • Utilizar la herramienta de búsqueda

    Usa palabras claves para encontrar la extensión que buscas

  • Elige extensiones compatibles con tu versión

    Una vez hayas encontrado la extensión que estabas buscando, asegúrate antes de descargarla que es compatible con la versión que utilizas.

Aporta tu colaboración

Para colaborar, sólo necesitas estar registrado a PEM

  • Autenticarse

    Si el sitio contiene un foro FluxBB y estás registrado en él, puedes autenticarte con el mismo usuario y contraseña.

  • Crear una cuenta

    Haz clic en Registrarse y cumplimenta la información requerida.
